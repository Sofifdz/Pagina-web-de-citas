# Pagina-web-de-citas

Este proyecto es una plataforma web de citas diseñada para conectar a personas con intereses y preferencias similares. 

## Instrucciones para Ejecutar la Aplicación

  1. Clonar el Repositorio: Abre tu terminal y ejecuta el siguiente comando -> git clone https://github.com/Sofifdz/Pagina-web-de-citas.git
     
  2. Instala las dependencias -> npm install
     
  3. Inicia el servidor -> npm start

## Tecnologias utilizadas

  1. React
  
  2. Node.js & Express.js
  
  3. PostgreSQL
